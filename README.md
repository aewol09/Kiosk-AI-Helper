# ☕ 키오스크 도우미 (Kiosk Helper) - Web App Version

고령층과 디지털 취약계층이 실제 카페 키오스크를 쉽고 직관적으로 학습할 수 있도록 돕는 웹 기반 키오스크 연습 프로그램입니다.
메인 백엔드 시스템은 Python Flask로 기동되며, 프론트엔드는 모던 HTML5, CSS3 및 JavaScript(Vanilla JS)로 설계되어 브라우저에서 편리하게 구동할 수 있습니다.

---

## 🌟 핵심 개편 사항

1. **완전한 웹 애플리케이션 전환**: 파이썬의 Tkinter 데스크톱 GUI에서 발생하는 리눅스 환경 폰트 렌더링 및 오디오 세그멘테이션 폴트(Segmentation fault) 문제를 완벽하게 회피하기 위해, 현대적인 웹 브라우저 기반 GUI로 전환하였습니다.
2. **동적 물리 이동 AI 바리스타 도우미**: 화면 속의 AI 도우미 캐릭터가 각 단계마다 사용자가 클릭해야 하는 단추(메뉴 카드, 옵션 확인, 결제 및 포장 단추 등)를 실시간 좌표 연산을 통해 **가장 알맞은 위치로 스르륵 미끄러지며(Glide) 이동하여** 말풍선으로 클릭 위치를 가리키고 안내합니다. 타겟 단추에는 깜빡이는 빨간색/노란색 고대비 테두리 가이드가 표시됩니다.
3. **음향 겹침(TTS) 방지 장치**: 가이드 음성 출력(Web Speech API Synthesis)이 재생되기 직전에 기존 목소리 출력을 즉시 소거(`speechSynthesis.cancel()`)하여, 사운드가 겹치거나 지연되어 웅성거리며 들리는 문제를 완벽하게 해결했습니다.
4. **실제 화면 구현 (코드 UI)**: 단순히 캡처 이미지를 배경으로 까는 방식 대신, 메가커피의 고유 컬러(다크 브라운, 주황색, 노란색) 및 컴포넌트(카테고리 탭, 그리드 카드, 장바구니 리스트, 옵션 모달, 포장 팝업 등)를 HTML/CSS 코드로 정밀하게 설계 및 구현하여 반응성이 높고 깔끔합니다.
5. **크로스플랫폼 브라우저 마이크 연동 (STT)**: 파이썬의 복잡한 PyAudio/PortAudio 드라이버 설치 없이 브라우저 자체 마이크(Web Speech Recognition API)를 사용하여 마이크 퍼미션만 주면 "아이스 아메리카노 주문해줘", "포장할래", "카드결제 해줘" 등의 목소리 인식이 즉시 활성화됩니다.
6. **학습 기록 대시보드 연동**: 연습 시간 및 실수가 저장되는 `history.json` 데이터가 메인 화면 대시보드의 누적 횟수, 평균 실수를 실시간 집계하여 로드됩니다.

## 📂 프로젝트 폴더 구조

```text
kiosk_ai/
├── app.py                  # Flask 웹 서버 메인 기동 파일 (Controller)
├── mega_menu.json          # 브랜드별 메뉴 및 가격 JSON 데이터
├── history.json            # 학습 이력 저장용 JSON (자동 생성)
├── requirements.txt        # 필요 파이썬 라이브러리 목록
├── README.md               # 프로젝트 안내서
├── spec.md                 # 기획 명세서
├── models/                 # 백엔드 데이터 모델 (M)
│   ├── menu_translator.py  # 외래어 설명 번역기 (Gemini/OpenAI & 로컬 사전)
│   └── history_manager.py  # 학습 기록 데이터 제어 및 통계
├── templates/              # HTML 템플릿 폴더
│   └── index.html          # 단일 페이지 애플리케이션 메인 마크업 (View)
└── static/                 # 정적 리소스 폴더
    ├── css/
    │   └── style.css       # 메가커피 테마, AI 글라이딩 이동 및 점멸 가이드 CSS
    └── js/
        └── app.js          # 키오스크 상태 머신 제어, AI 캐릭터 연산, browser STT/TTS JS
```

## 🚀 기동 및 접속 방법

### 1. 의존성 패키지 설치

파이썬 환경에서 Flask만 설치되어 있다면 별도의 마이크 드라이버 컴파일 없이 구동 가능합니다.

```bash
pip install -r requirements.txt
```

### 2. 서버 실행

```bash
python3 app.py
```

기동이 완료되면 터미널에 `* Running on http://127.0.0.1:5000` 문구가 노출됩니다.

### 3. 브라우저로 접속

Chrome, Edge, Safari 등 최신 웹 브라우저를 열고 다음 주소에 접속하여 키오스크 연습 프로그램을 사용합니다.

👉 **[http://127.0.0.1:5000/](http://127.0.0.1:5000/)**

> 💡 **안내**: 마이크 권한 요청 팝업이 나타날 경우 **'허용'** 단추를 눌러주시면 음성 인식 주문 기능을 사용할 수 있습니다. 마이크 장치가 없거나 가동이 어려울 경우 우측 패널의 **'말하기 예시 단추'**들을 누르시면 간편하게 모의 주문을 시험할 수 있습니다.
