너는 시니어 Python 개발자이자 UI/UX 개발자이다.

"키오스크 도우미 앱(Kiosk Helper)"을 Python으로 구현하라.

# 프로젝트 목표

고령층과 디지털 취약계층이 실제 카페 키오스크를 연습할 수 있도록 돕는 AI 기반 키오스크 학습 프로그램을 만든다.

실제 결제는 하지 않는다.

주문 연습 및 사용법 학습만 제공한다.

---

# 개발 환경

Python 3.12

GUI : CustomTkinter

음성 인식 : SpeechRecognition

음성 출력 : pyttsx3

AI 메뉴 설명 : OpenAI API 또는 Gemini API 사용 가능하도록 구조화

데이터 저장 : JSON

이미지 처리 : Pillow

---

# 핵심 기능

## 1. 메인 화면

* 서비스 로고 표시
* 시작하기 버튼
* 사용 방법 버튼
* 종료 버튼

---

## 2. 브랜드 선택

초기 버전

* 메가커피

추후 확장 가능하도록 구조 설계

브랜드 데이터는 JSON으로 분리

---

## 3. 음성 주문 기능

사용자가

"아이스 아메리카노 주문하고 싶어"

라고 말하면

STT를 통해 텍스트 변환

메뉴명 추출

주문 시나리오 생성

---

## 4. AI 도우미 캐릭터

화면 우측 하단 고정

기능

* 말풍선 출력
* 현재 단계 설명
* 다음 클릭 위치 안내

예시

"커피 메뉴를 눌러주세요"

"아이스 아메리카노를 선택하세요"

"주문담기 버튼을 누르세요"

---

## 5. 실제 메가커피 키오스크 학습 모드

첨부된 메가커피 키오스크 흐름을 그대로 구현

주문 단계

1단계 메뉴 선택

2단계 옵션 선택

* 텀블러
* 샷 추가
* 당도
* 토핑

3단계 주문 확인

4단계 매장 이용 / 포장 선택

5단계 결제수단 선택

6단계 결제 완료

---

## 6. 클릭 가이드

AI 캐릭터가

* 버튼을 강조 표시
* 깜빡이는 테두리 표시
* 화살표 표시

하여 사용자가 눌러야 할 위치를 안내

---

## 7. 메뉴 번역기

낯선 메뉴 클릭 시

예시

아인슈페너

↓

"커피 위에 달콤한 크림을 올린 음료"

플랫화이트

↓

"우유가 들어간 진한 라떼"

처럼 쉬운 한국어 설명 표시

AI API를 사용하지 못하는 경우에도 동작하도록

사전(Dictionary) 기반 fallback 구현

---

## 8. 학습 기록

JSON 저장

저장 항목

* 연습 날짜
* 브랜드
* 주문 메뉴
* 소요 시간
* 실수 횟수

---

# UI 요구사항

중장년층 대상

반드시 적용

* 큰 글씨

* 고대비 색상
* 단순한 화면
