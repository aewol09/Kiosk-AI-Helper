# [설계 보고서] AI 기반 키오스크 교육 앱: 키오스크 헬퍼 (Kiosk Helper)

---

## 1. 사용자 시나리오

본 교육 앱의 주 타깃층인 일반 성인 사용자와 디지털 취약계층(65세 이상 고령자)의 실제 사용 경험을 구체화하기 위해 두 가지 사용자 시나리오를 정의합니다.

### 시나리오 1: 일반 성인 사용자 (디지털 기기에 익숙하나, 새로운 키오스크나 복잡한 옵션 선택에서 교육/연습이 필요한 경우)

*   **상황 (Situation)**
    *   30대 직장인 김민수 씨는 바쁜 출근 시간대에 메가커피 매장을 방문했습니다. 평소 모바일 앱 주문을 주로 사용했으나, 당일 네트워크 오류로 인해 매장 내 설치된 실제 키오스크를 직접 조작해야 하는 상황에 놓였습니다. 뒤에 줄을 서 있는 다른 대기 고객들을 보고 심리적 압박감을 느끼며, 미리 키오스크 주문 프로세스(특히 복잡한 텀블러 할인 및 개인 옵션 조합)를 집이나 이동 중에 빠르고 완벽하게 연습해보고자 앱을 실행합니다.
*   **목표 (Goal)**
    *   실제 메가커피 키오스크와 동일한 환경에서 "아메리카노 샷 추가 및 개인 텀블러 할인 옵션"을 적용하여 주문하고 결제 및 적립까지 마치는 전 과정을 완벽히 마스터하는 것입니다.
*   **행동 (Action)**
    *   앱을 켜고 '메가커피' 학습 모드를 선택합니다.
    *   음성 입력 화면에서 "아메리카노 텀블러에 담아서 샷 추가해줘"라고 입력하여 AI 도우미가 가이드하는 옵션 흐름을 파악합니다.
    *   실제 키오스크 화면과 1:1로 매칭되는 교육 화면에서 텀블러 선택(0원), 샷 추가(+600원) 옵션을 순서대로 활성화하고 가상 장바구니에 담습니다.
    *   최종 화면에서 '먹고가기'와 '카드결제' 버튼을 눌러 승인 완료 단계까지 막힘없이 진행합니다.
*   **결과 (Result)**
    *   메가커피 키오스크의 전체 주문 흐름(메뉴 -> 옵션 -> 장바구니 -> 할인/결제수단 선택 -> 적립/승인)을 완벽히 숙지하게 되었습니다. 실제 오프라인 매장 키오스크 앞에서도 당황하거나 뒤 대기자들의 눈치를 보지 않고, 30초 만에 원하는 옵션을 정확히 선택하여 주문에 성공했습니다.

---

### 시나리오 2: 65세 이상 고령자 사용자 (디지털 기기 및 키오스크 인터페이스에 심한 두려움을 가진 디지털 취약계층)

*   **상황 (Situation)**
    *   72세 박영자 할머니는 손주가 좋아하는 "시원한 아메리카노"를 직접 사주고 싶어 집 앞 메가커피 매장 앞에 섰습니다. 하지만 글씨가 너무 작고 복잡한 영어 카테고리(디카페인, 슈페너 등), 눈부신 화면 구성과 낯선 안내 문구 때문에 키오스크 조작에 실패한 경험이 있어 매장 밖에서 망설이고 있습니다. 집에서 자녀가 설치해 준 '키오스크 헬퍼' 앱을 열고 AI 캐릭터의 음성 지원을 받으며 연습해보기로 합니다.
*   **목표 (Goal)**
    *   "아이스 아메리카노 한 잔 주문하기"를 AI 도우미 캐릭터의 친절한 비주얼 가이드와 음성 안내를 받으며 천천히 완벽하게 성공하는 것입니다.
*   **행동 (Action)**
    *   시작 화면에서 아주 큰 글씨로 쓰인 '시작하기' 버튼을 누릅니다.
    *   음성 인식 마이크 아이콘을 누르고 "아이스 아메리카노 주문하고 싶어"라고 말합니다.
    *   우측 하단에 나타난 친절한 AI 도우미 캐릭터 '헬피'가 따뜻한 목소리로 "화면 가운데에 있는 '(ICE)아메리카노' 메뉴를 눌러보세요"라고 말하며, 해당 버튼 주변을 반짝이는 노란색 테두리와 움직이는 손가락 애니메이션으로 강조해 줍니다.
    *   옵션 선택 창이 떴을 때 당황하지 않도록 캐릭터가 "추가할 옵션이 없다면 우측 하단의 노란색 '주문담기' 버튼을 눌러주세요"라고 안내해 줍니다.
    *   최종 확인 단계에서 '먹고가기'와 '포장하기'의 차이를 소리로 들으며 '포장하기'를 선택하고, 결제 단계에서 가상 카드를 넣는 일러스트 애니메이션을 따라 카드를 삽입해 봅니다.
*   **결과 (Result)**
    *   매번 자녀들에게 부탁하거나 점원에게 눈치 보며 주문하던 영자 할머니가 AI 캐릭터의 단계별 맞춤 밀착 케어를 통해 스스로 음료를 주문하는 성공 경험을 얻었습니다. 기기에 대한 두려움이 극복되었으며, "다음에는 진짜 매장에 가서 직접 주문해 봐야겠다"라는 강한 자신감을 가지게 되었습니다.

## 2. 정보구조도 (Information Architecture, IA)

전체 애플리케이션의 구조는 사용자의 학습 흐름을 끊김 없이 지원하고, 필요시 언제든지 이전 단계나 설정으로 접근할 수 있도록 직관적인 트리 구조로 설계되었습니다.

```
홈 (Main Home)
├─ 카페 선택 (Cafe Selection)
│   └─ 메가엠지씨커피 (Mega MGC Coffee - 활성화)
│       ├─ 음성 입력 및 가상 미션 설정 (Voice Input / Mission)
│       │   └─ "아이스 아메리카노 주문하기" 미션 생성
│       └─ 실제 주문 연습 (Kiosk Simulation)
│           ├─ 1단계: 메뉴 선택 (Menu Selection)
│           │   ├─ 신메뉴 / 추천메뉴 / 커피 / 디카페인 / 음료 / 티 / 푸드 / 상품 (카테고리 탐색)
│           │   ├─ 상품 목록 (예: ICE 아메리카노, 메가리카노 등)
│           │   ├─ 장바구니 조회 (수량 변경 및 삭제)
│           │   └─ 결제하기 (다음 단계 진입)
│           ├─ 2단계: 옵션 선택 (Option Selection - Modal)
│           │   ├─ 텀블러 선택 (개인컵 사용 여부)
│           │   ├─ 샷 선택 (연하게 / 샷 추가 / 2샷 추가)
│           │   ├─ 저당 스테비아 슈가 추가
│           │   ├─ 당도 선택 (연유 추가 / 저당 스테비아 추가)
│           │   ├─ 토핑 선택 (휘핑 추가)
│           │   ├─ MGC 추천 옵션 (아메리카노 꿀팁 옵션)
│           │   └─ 하단 제어 (취소 / 주문담기)
│           ├─ 3단계: 최종 확인 및 장소 선택 (Order Confirmation - Modal)
│           │   ├─ 주문 세부 내역 재확인 (메뉴명, 수량, 금액)
│           │   ├─ 일회용컵 사용 불가 경고 문구 노출
│           │   ├─ 먹고가기 (다회용컵) 선택
│           │   └─ 포장하기 (일회용컵) 선택
│           ├─ 4단계: 할인 및 결제수단 선택 (Payment & Discount - Modal)
│           │   ├─ STEP 1: 할인수단 선택 (KT VIP, T멤버십, CJ ONE, SKT 우주패스 등)
│           │   └─ STEP 2: 결제수단 선택 (카드결제, 앱카드, 네이버페이, 카카오페이 등)
│           ├─ 5단계: 적립 및 결제 프로세스 (Stamp & Execution - Modal/Overlays)
│           │   ├─ 스탬프 적립 선택 (010 휴대폰 번호 가상 키패드 입력)
│           │   ├─ 신용카드 가상 IC 카드 투입 애니메이션 및 승인 요청
│           │   └─ 정상 승인 및 영수증 출력 여부 선택 (출력 / 미출력)
│           └─ 6단계: 완료 화면 (Completion Screen)
│               ├─ 주문 번호 발행 시뮬레이션
│               ├─ 완료 피드백 및 가상 미션 성공 축하 연출
│               └─ 메인 홈으로 돌아가기 / 다시 연습하기
├─ 사용 방법 (Tutorial Guide)
│   ├─ 음성 인식 사용법 안내
│   └─ AI 도우미 '헬피' 조작법 및 말풍선 가이드 이해
└─ 환경 설정 (Settings)
    ├─ 글자 크기 조절 (보통 / 크게 / 아주 크게)
    ├─ 음성 안내 속도 조절 (0.8x / 1.0x / 1.2x)
    ├─ 화면 고대비 테마 (ON/OFF)
    └─ AI 도우미 캐릭터 변경 및 볼륨 조절
```

## 3. 주요 화면 설계

키오스크 헬퍼의 직관적이고 실감 나는 주문 교육 환경을 제공하기 위해 **총 8개 주요 화면**을 정의하고 설계 명세를 구체화합니다.

### 화면 1: 홈 (Home)
*   **목적**
    *   앱 최초 실행 시 사용자를 맞이하며, 명확하고 단순한 진입점을 제공하여 조작의 두려움을 최소화합니다.
*   **주요 기능**
    *   주문 연습 시작하기 진입
    *   처음 방문한 사용자를 위한 쉬운 '사용 방법' 애니메이션 튜토리얼 링크
    *   전체 글자 크기 및 오디오 볼륨을 빠르게 맞출 수 있는 간이 설정 제공
*   **입력 데이터**
    *   [터치] 시작하기 버튼, 사용 방법 버튼, 설정 버튼
*   **출력 데이터**
    *   웰컴 음성 가이드 ("안녕하세요! 키오스크 헬퍼와 함께 쉽고 재미있게 주문을 연습해 봐요.")
    *   가장 선명하고 큰 UI 카드 컴포넌트
*   **UI 구성요소**
    *   **헤더**: 앱 로고, 캐릭터 아이콘, 톱니바퀴 모양 설정 버튼 (크기 최소 48x48dp 이상 확보)
    *   **바디**: 화면의 60% 이상을 차지하는 거대한 노란색 `[시작하기]` 버튼, 은은한 파란색 `[사용 방법]` 버튼 (고대비 및 친근한 둥근 둥글 형태)
    *   **푸터**: 보조 캐릭터 일러스트 및 따뜻한 격려 문구 ("당겨서 연습을 시작해보세요!")

---

### 화면 2: 카페 선택 (Cafe Selection)
*   **목적**
    *   사용자가 일상에서 직면할 특정 프랜차이즈 브랜드를 직접 선택하여 현실감 높은 1:1 맞춤 교육 환경을 준비합니다.
*   **주요 기능**
    *   지원 가능한 프랜차이즈 카페 목록 노출 (초기 버전: 메가커피 활성화, 타 브랜드는 '준비 중' 표시)
    *   카페 검색 기능 및 고령자를 위한 대표 로고 중심의 그리드 배치
*   **입력 데이터**
    *   [터치] 브랜드 카드 선택 (메가커피), 이전으로 가기 버튼
*   **출력 데이터**
    *   브랜드 선택 안내 가이드 음성 및 시각적 피드백
*   **UI 구성요소**
    *   **헤더**: 뒤로 가기 타이틀 ("어느 카페에서 주문해 볼까요?")
    *   **브랜드 목록**: 메가MGC커피 로고와 한글 텍스트가 아주 크고 선명하게 명시된 카드형 레이아웃 (노란색 아이덴티티 강조), 비활성화된 그레이아웃 카드 (스타벅스, 투썸플레이스, 이디야 등 - '준비중' 엠블럼 부착)
    *   **AI 도우미 말풍선**: 우측 하단에서 "가장 대중적인 '메가커피'부터 천천히 시작해 보아요!" 음성 송출

---

### 화면 3: 음성 입력 (Voice Input)
*   **목적**
    *   원하는 음료 및 목적을 텍스트 입력 없이 오직 편안한 말 한마디로 주문 가상 미션으로 변환하는 혁신적 자연어 인터페이스 화면입니다.
*   **주요 기능**
    *   실시간 마이크 입력 인식 (STT: Speech To Text Engine 활용)
    *   목소리가 녹음되는 시각적 이퀄라이저 표시 및 텍스트 실시간 디스플레이
    *   인식된 텍스트를 분석하여 맞춤 미션(예: "아이스 아메리카노 1개 담기") 자동 생성 및 시뮬레이터 연결
*   **입력 데이터**
    *   [음성] 사용자 음성 데이터 ("아이스 아메리카노 한 잔 주문할래", "따뜻한 라떼 한 잔 줘" 등)
    *   [터치] 마이크 아이콘 버튼, 직접 텍스트로 치기 링크
*   **출력 데이터**
    *   화면에 표시되는 가청 텍스트 ("아이스 아메리카노 주문하고 싶어")
    *   AI 가이드 음성 ("좋습니다! 아이스 아메리카노를 주문하는 미션을 함께 시작해 봐요. 제가 깜빡이는 곳을 알려 드릴게요.")
*   **UI 구성요소**
    *   **가운데 영역**: 거대한 마이크 애니메이션 서클 버튼 (활성화 시 펄스 애니메이션 적용), 마이크 주변 "말씀해 주세요" 안내 문구
    *   **텍스트 변환 피드백**: 실시간으로 음성이 한글 자막처럼 아주 크고 굵게 타이핑되는 텍스트 상자
    *   **가이드 패널**: 추천 발화 예시 노출 ("아메리카노 시원한 거 한 잔 줘", "달콤한 카페라떼 포장해 줘")

---

### 화면 4: 메뉴 선택 (Menu Selection - 실제 메가커피 이미지 1 대응)
*   **목적**
    *   실제 메가커피 키오스크 화면을 1:1로 완벽히 재현하여 오프라인 사용 시 이질감을 극 제로화하며, 가상 장바구니에 음료를 담는 연습을 진행합니다.
*   **주요 기능**

## 4. UI/UX 설계 명세 (실제 이미지 기반 입체 분석)

실제 메가커피의 복잡하고 혼란스러운 정보 구조와 색상을 분석하여, 고령자와 취약계층의 관점에서 이들을 완벽하게 케어할 수 있는 핵심적인 UI/UX 설계 원칙을 수립하였습니다.

### 4.1. 접근성 (Accessibility) 강화 전략

*   **컴포넌트 크기 및 터치 타깃 확장**
    *   실제 이미지 속 키오스크 카테고리 탭과 그리드는 모바일 화면에 그대로 이식할 시 고령층이 터치하기에 지나치게 좁고 빽빽합니다.
    *   연습용 모바일 앱 내부의 가상 키오스크 화면은 **최소 48dp x 48dp 이상의 절대 터치 타깃 영역**을 강제로 확보하며, 터치 유효 범위를 버튼 바깥까지 약 20% 마진 확장하여 손가락이 미세하게 떨리더라도 오클릭 없이 정상 작동하도록 설계합니다.
*   **극대화된 가독성 및 글씨 크기 제어**
    *   실제 키오스크 상의 옅은 그레이 계열 가이드 폰트는 저시력 고령자에게 보이지 않습니다. 본 앱은 메인 텍스트에 **24sp 이상의 고대비 볼드체(Noto Sans KR 등 검증된 폰트)**를 사용하며, 서브 텍스트 역시 최소 16sp 이상으로 고정합니다.
    *   상시 사용 가능한 3단계 글자 크기 스케일링 조절기(보통, 크게, 아주 크게)를 모든 주요 연습 오버레이 상단에 플로팅 상태로 탑재합니다.
*   **고대비 컬러 배색 (High-Contrast Theme)**
    *   시각적 정보 인식률을 극대화하기 위해 흰색 배경에 연한 노란색 폰트를 사용하는 실수의 조합을 피합니다.
    *   **배경 대비 최소 4.5:1 이상의 명도 대비**를 철저히 지키며, 가상 키오스크 위의 글씨는 검정색 배경의 노란색 텍스트 또는 흰색 배경의 진한 다크 브라운 텍스트로 치환하여 인쇄물 수준의 가독성을 제공합니다.

### 4.2. 사용성 (Usability) 강화 및 오작동 안전장치

*   **직관적인 글로벌 진행상황 인디케이터 (Stepper)**
    *   현재 화면이 `메뉴 선택` -> `옵션 선택` -> `매장 선택` -> `결제수단 선택` -> `카드 결제` 중 어느 지점에 속해있는지, 화면 맨 위에 커다란 숫자가 적힌 단계 스텝 바를 상시 배치합니다.
    *   "총 5단계 중 3단계 진행 중입니다"와 같은 현재 진행 멘트를 제공하여 사용자가 길을 잃지 않도록 심리적 안정을 도모합니다.
*   **실수 감지 및 자동 복구 (Anti-Error Safeguard)**
    *   연습 도중 미션 범위를 완전히 벗어난 엉뚱한 버튼(예: 아메리카노 미션 중에 뜬금없는 디저트 상품 탭 등)을 연속 3회 이상 터치할 경우, 앱이 오작동으로 판단하여 진동 피드백과 함께 AI 캐릭터 헬피가 가볍게 흔들리며 도움 팝업을 제공합니다.
    *   "혹시 메인 화면으로 돌아가고 싶으신가요? 지금은 아메리카노를 담을 차례예요"라며 부드러운 올바른 길 찾기 로직을 발동합니다.
*   **가상 결제/적립 훈련 시스템**
    *   적립 및 결제 실패율이 가장 높다는 실제 이미지 5, 6 단계를 완벽히 해결하기 위해, 스마트폰의 NFC 가상 영역을 매핑하거나 모의 카메라로 연습 카드를 촬영하는 등 일련의 실물 삽입 제스처 게임화(Gamification) 방식을 도입하여 인지 능력을 크게 상승시킵니다.

### 4.3. 실감형 AI 도우미 캐릭터 '헬피(Helpy)'의 연출 상세

*   **친근한 시각적 배치와 인터랙션**
    *   도우미 캐릭터 '헬피'는 항상 화면의 **우측 하단**에 반투명한 플로팅 구 형태로 상주하며, 키오스크 본체 레이어를 가리지 않도록 사용자의 시야 흐름에 따라 좌측이나 중앙으로 자연스럽게 순간이동하여 유동적으로 자리를 피합니다.
*   **정밀한 하이라이팅 가이드 및 화살표 애니메이션**
    *   실제 눌러야 하는 특정 타깃 버튼 영역(예: 이미지 1의 (ICE)아메리카노 카드 또는 이미지 3의 '포장하기') 주변을 감싸는 **노란색/네온 그린빛 테두리가 맥박 치듯(Pulsing) 깜빡이는 연출**을 적용합니다.
    *   동시에 귀여운 캐릭터 3D 손가락 그래픽이 해당 버튼을 반복해서 누르는 듯한 화살표 바운스 애니메이션을 오버레이로 제공하여 주의력을 한곳으로 집중시킵니다.
*   **따뜻한 오디오 보이스 가이드 (TTS)**
    *   기계적이고 차가운 기계 합성음 대신, 전문 성우 톤의 다정하고 느릿하며 명확한 어조의 오디오 피드백을 전달합니다.
    *   단어를 너무 빠르게 발음하지 않고, 어절 단위로 명확히 끊어 읽는 음성 합성 프로필을 사용하여 고령층 사용자가 단어 하나하나를 귀로 정확히 인지하도록 제어합니다.
*   **실시간 클릭 피드백 연출**
    *   가이드에 따라 올바른 터치 포인트를 올바르게 클릭했을 때, 캐릭터가 빙글 돌거나 머리 위로 하트 마크를 띄우며 "아주 잘하셨어요! 정확하게 누르셨습니다!"라는 긍정적인 즉각적 피드백 효과음과 애니메이션을 출력하여 동기부여와 성취감을 극대화합니다.
